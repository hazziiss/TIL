> # 2022.12.27 학습) GIT 기본 문법

## `♪ (.git) 항목 찾는 명령어`
		$ git init

## `♪ 숨김 폴더 확인 명령어`
		$ ls -al

## `♪ 상태 보는 명령어 (1통)`
		$ git status

## `♪ 1통에서 2통으로 넘기는 명령어`
		$ git add .

## `♪ 2통에서 3통으로 넘기는 명령어`
		$ git commit -m '수정되었습니다'		

## `♪ 버전 확인 명령어`
		$ git log
